## KEY TOPICS

### Print-on-Demand Pipeline with AI

| Step | Tool | Action |
|------|------|--------|
| Niche discovery | Claude / ChatGPT | Prompt for underserved micro-niches (night shift nurses, ADHD adults, female electricians) |
| Bestseller research | Etsy | Filter by "bestseller" in URL, screenshot top 8 designs |
| Design analysis | Claude | Paste screenshot, ask AI to reverse-engineer success factors (emotional triggers, identity, humor) |
| Concept generation | Claude | Output ~15 original product concepts formatted as image gen prompts |
| Design creation | Nano Banana 2 (gemini.google.com) | Paste prompt with "3600 x 4800" dimension spec for print quality |
| Store setup | Hostinger | AI website builder, customize in 1-2 hours |
| Fulfillment | Printful | Connect to Hostinger, drag-and-drop product setup |
| Marketing | TikTok / Instagram | Organic short-form video (film yourself using the product) |

### Why NOT Etsy
- Oversaturated with AI-generated designs
- Etsy algorithm favors established sellers
- Better margins and control on independent store

### Pricing Model
- Printful base cost (production) + markup = retail price
- Example: retail $15, profit ~$7.50 per sale
