### Claude Code Integration
1. Fork and clone the repo
2. Run setup command to transform agents into sub-agent directories
3. Reference agents by name or tag the .md file in prompts
4. Multi-agent orchestration: launch multiple sub-agents in parallel
5. Customization: change personality, colors, tools, model, add persistent memory (`memory: project`)
