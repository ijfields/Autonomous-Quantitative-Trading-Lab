## What This Guide Covers

A walkthrough of all major Claude updates from February 2026 — new models, free plan features, agent teams, Co-work plugins, Claude in Chrome — and how to start using each one.

---

## Update 1: Sonnet 4.6 (Default Model)

- **Released:** Feb 17, 2026
- **What:** Mid-tier model, now default on free plan
- **Best for:** Coding, tool use, agentic work, office tasks
- **Key insight:** Nearly matches Opus on most day-to-day tasks
- **Action:** If you're on the free plan, you're already using it

---

## Update 2: Opus 4.6 (Flagship)

- **Released:** Feb 5, 2026
- **What:** Most advanced model for hard problems
- **Best for:** Complex reasoning, novel problem solving, long agentic sessions
- **Action:** Use for tasks where Sonnet falls short — deep reasoning, multi-step problems

---

## Update 3: Free Plan Features (Feb 11)

| Feature | What You Can Do |
|---------|----------------|
| File creation | Create Word docs, Excel, PowerPoint, PDFs |
| Connectors | Link Google Workspace, Canva, Notion, Slack |
| Skills | Create reusable mini-playbooks for repeated workflows |

**Action:** Go to claude.ai → try creating a file or connecting a tool

---

## Update 4: Voice Mode on Desktop

- Now available on desktop app and web (was mobile-only)
- **Action:** Open Claude desktop app → look for microphone icon → speak your prompts

---

## Update 5: Compaction (Longer Conversations)

- When conversation approaches context limit, Claude auto-summarizes older messages
- Recent messages stay fully intact
- Available on all plans including free
- **Action:** Nothing to do — it happens automatically

---

## Update 6: Agent Teams (Claude Code)

- Spin up multiple AI agents working in parallel
- One team lead coordinates; teammates work independently
- Teammates can message each other directly

**When to use:**
- Parallel exploration (testing multiple approaches)
- Independent pieces (building separate features)

**When NOT to use:**
- Sequential work where one step depends on the previous
- Token-sensitive projects (uses significantly more tokens)

**Status:** Research preview

---

## Update 7: Co-work + Plugins

1. Open Claude Desktop App
2. Click the **Co-work** tab
3. Browse and install plugins (sales, legal, marketing, etc.)
4. Give Claude a goal → it does the work autonomously
5. Use **Plugin Create** to build custom plugins

**Available on:** All paid plans (Pro $20, Max $200, Team, Enterprise)

---

## Update 8: Claude in Chrome

1. Install the **Claude Chrome Extension** from Chrome Web Store
2. Pair with Co-work for web-based tasks
3. Claude can: read pages, click, fill forms, navigate apps
4. **Available to:** All paid plan subscribers
