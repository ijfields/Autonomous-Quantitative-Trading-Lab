## What This Guide Covers

How to set up and use Perplexity Computer, a cloud-based AI agent platform that requires no local setup, as an alternative to OpenClaw for building automated workflows and dashboards.

---

## Step 1: Choose Your Plan

| Plan | Price | Credits | Best For |
|------|-------|---------|----------|
| Pro | $20/month | 4,000 credits | Light usage, testing |
| Max | $200/month | 10,000 credits | Production workflows |

1. Go to Perplexity.ai or use the Comet Browser
2. Subscribe to Pro or Max plan
3. No local installation, no server setup required

---

## Step 2: Understand the Model Orchestrator

1. Perplexity Computer does NOT use a single AI model
2. It has a "model orchestrator" that assigns each subtask to the best available model:
   - Claude Opus for coding tasks
   - Nano Banana for image generation
   - Other specialized models for research, data processing, etc.
3. Tasks run in parallel for speed
4. The system does its own QA — it uses a built-in browser to verify outputs before delivering

---

## Step 3: Build Your First Dashboard

1. Describe what you want in natural language (e.g., "Create an S&P 500 earnings dashboard showing latest quarterly results")
2. Perplexity Computer will:
   - Research the data sources
   - Build the interface
   - Populate with live data
   - Verify the output
3. Review and iterate — request changes conversationally

---

## Step 4: Set Up Scheduled Tasks

1. Create a recurring task (e.g., "Every morning at 7 AM, compile a personal briefing with my calendar, news, and priorities")
2. Perplexity Computer runs scheduled tasks in the cloud 24/7
3. No local machine needs to be on — everything runs server-side
4. Results delivered to your inbox or dashboard

---

## Step 5: Connect Apps via Connectors

1. Browse available "connectors" (similar to Zapier integrations)
2. Connect the apps you use: email, calendar, project management, social media, etc.
3. Build workflows that span multiple apps
4. Example: YouTube → Instagram content repurposing pipeline

---

## Step 6: Use Voice Mode for Complex Requests

1. Activate voice mode for hands-free workflow creation
2. Describe multi-step workflows conversationally
3. Perplexity Computer builds, tests, and deploys them
4. Iterate verbally — no typing required

---

## Step 7: Compare with OpenClaw (When to Use Which)

| Use Case | Perplexity Computer | OpenClaw |
|----------|-------------------|----------|
| Quick setup, no tinkering | Best choice | Requires setup |
| Cloud-based, no local machine | Yes | No (needs local/VPS) |
| Security-sensitive workflows | Cloud-managed | Tokens in plain text risk |
| Full customization, open-source | No | Yes |
| Cost | $20-$200/month | Free (but needs compute) |
| Developer-level control | Limited | Full control |

---

## Key Takeaway

> Perplexity Computer is the "plug-and-play" alternative to OpenClaw — zero setup, zero security concerns, cloud-native 24/7 operation. Its model orchestrator automatically selects the best AI for each subtask. Best for business owners who want results without infrastructure complexity. The trade-off: less customization than OpenClaw's open-source flexibility.
