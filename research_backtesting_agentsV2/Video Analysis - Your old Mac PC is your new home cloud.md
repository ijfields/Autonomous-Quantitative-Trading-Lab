## EXECUTIVE SUMMARY

Oliur demonstrates repurposing an old Mac (Mini, MacBook Air, etc.) as a self-hosting home cloud using Docker Desktop + Umbrel OS. The setup replaces cloud subscriptions (Google Drive, iCloud, Netflix) with self-hosted alternatives. Key apps: Nextcloud (files), Immich (photos), Home Assistant (smart home), Plex/Jellyfin (media), Tailscale (remote access), and OpenClaw. Emphasizes the 3-2-1 backup rule and external storage expansion for Mac's limited internal storage.

---

## KEY TOPICS

### Setup Process
1. Install Rosetta (for Apple Silicon Macs): `softwareupdate --install-rosetta`
2. Download and install Docker Desktop for Mac (Apple Silicon version)
3. Run Umbrel OS via Docker CLI command (GUI search didn't work — used Claude to troubleshoot)
4. Open Umbrel web interface (localhost), create local account

### Available Self-Hosted Apps

| App | Replaces |
|-----|----------|
| Nextcloud | Google Drive / Dropbox |
| Immich | iCloud Photos |
| Home Assistant | Smart home hub (Hue, Govee, etc.) |
| Plex / Jellyfin | Netflix (self-hosted media) |
| Tailscale | VPN for remote access |
| OpenClaw | AI agent platform |

### Storage and Backup
- External SSD or multi-bay HDD enclosure for expanded storage
- **3-2-1 backup rule:** 3 copies, 2 different devices, 1 offsite location
- Self-hosting alone is NOT sufficient backup for irreplaceable data

## ACTIONABLE TAKEAWAYS

1. Old Mac Mini or MacBook Air can become a home server with Docker + Umbrel in under 20 minutes
2. Replace cloud subscriptions with self-hosted alternatives
3. Always follow 3-2-1 backup rule — self-hosting alone isn't safe
4. External storage (SSD or RAID enclosure) practically required given Mac storage limits
5. Tailscale enables secure remote access from anywhere
