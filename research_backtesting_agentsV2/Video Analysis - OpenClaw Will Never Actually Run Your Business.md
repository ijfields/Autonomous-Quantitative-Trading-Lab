## Key Concepts

### Wrapper vs. Harness
- **Wrapper (old):** Thin layer around a model — basic functionality
- **Harness (new):** Robust system giving the model "arms" — file reading, web search, sub-agents, deep research
- Claude Code = billion-dollar harness by Anthropic; OpenClaw = indie developer harness attempting to replicate it
- Analogy: "You've taken a Ferrari engine and put it in a Honda chassis"

### Scaling Curves
- OpenClaw: Quick initial results, but performance **plateaus** as complexity increases
- Claude Code: Higher setup barrier, but **compounding** benefits with no ceiling found yet

### Context OS
- Liam's term for his Claude Code setup: structured memory + documentation + full database with all business data
- More comprehensive than OpenClaw's opaque memory system
- Pull from all data sources into one queryable place

### Feature Comparison
| Feature | OpenClaw | Claude Code |
|---------|----------|-------------|
| Telegram/WhatsApp | Built-in | Custom setup (more powerful, fully customizable) |
| Cron jobs | Built-in (confusing) | Custom setup (layers, fully controllable) |
| Memory | Opaque, hard to maintain | Structured Context OS + database |
| Web search | Third-party plugin | Native + deep research agents (trained, better) |
| Model selection | Any model | Haiku (cheap tasks), Sonnet (medium), Opus (complex) |
| Cost | Varies by model | $50-$200/mo Anthropic plan |
