## KEY TOPICS

### Model Updates

#### Sonnet 4.6 (Feb 17)
- Mid-tier model, now default on free plan
- Major upgrade for coding, tool use, agentic work
- Nearly matches Opus on: real-world software engineering, computer use, office workflows
- Faster and cheaper than Opus
- "The smaller sibling is outgrowing its role"

#### Opus 4.6 (Feb 5)
- Most advanced Claude model
- Better instruction following over long conversations
- Does not give up on hard problems (pushes through vs. surface-level answers)
- Sustains agentic tasks longer and more reliably
- Still leads on: deepest reasoning, novel problem solving, complex multi-step reasoning

### Free Plan Expansion (Feb 11)
- **File creation:** Word docs, Excel, PowerPoint, PDFs
- **Connectors:** Google Workspace, Canva, Notion, Slack, etc.
- **Skills:** Reusable mini-playbooks (define a process, Claude remembers and applies it)

### Voice Mode
- Now available on desktop app and web (previously mobile only)
- Speak prompts instead of typing
- Hear responses back

### Context Window & Compaction
- 1M token context window in beta (API, high-usage developers)
- **Compaction:** When conversation approaches limit, Claude auto-summarizes older messages
- Recent messages stay fully intact
- Available on free plan

### Claude Code Improvements
- Explores bigger codebases
- Writes more code per prompt
- Better at complex multi-file tasks
- Big jump in real-world engineering benchmarks
- Plan mode for better results
- Multi-agent support (launches multiple agents simultaneously)

### Agent Teams
- Multiple AI agents work on different parts of a task in parallel
- One session acts as team lead; teammates work independently
- Teammates can communicate directly with each other
- Different from sub-agents (which run within a single session)
- Uses significantly more tokens — be intentional
- Best for: parallel exploration, independent pieces
- Research preview status

### Co-work + Plugins
- Claude Code for non-technical people
- Give Claude a goal → it does the work (reading files, organizing, drafting, research)
- Available on all paid plans (Pro, Max, Team, Enterprise)
- 11 starter plugins at launch (sales, legal, support, data, marketing, etc.)
- New plugins for HR, finance, engineering
- Plugin Create plugin for building custom plugins

### Claude in Chrome
- Chrome extension for browser control
- Reading pages, clicking, filling forms, navigating apps
- Pair with Co-work for web-based tasks
- Download separately from Chrome Web Store

### Live Demo: Recipe Website
- Single prompt → complete Next.js + ShadCN recipe website
- Used plan mode (Shift+Tab) for better results
- Launched multiple agents for parallel component building
- ~10 minutes to complete build
- Deployed to Hostinger via GitHub import
