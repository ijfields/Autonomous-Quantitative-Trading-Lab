# Step-by-Step Guide: Using Claude in Chrome as an AI Agent

**Source:** Moon Dev (YouTube)
**Video ID:** TzZVeh4gHbk
**Upload Date:** 2026-03-07

---

## What This Guide Covers

How to set up and use Claude in Chrome (the Chrome extension) as a browser-based AI agent — running multiple instances for different tasks like research, trading analysis, and data collection.

---

## Prerequisites

- Paid Claude plan (Pro, Max, Team, or Enterprise)
- Google Chrome browser
- Claude Chrome Extension (from Chrome Web Store)

---

## Step 1: Install Claude Chrome Extension

1. Open Chrome Web Store
2. Search for "Claude" extension
3. Install and add to Chrome
4. Sign in with your paid Claude account

---

## Step 2: Launch Your First Instance

1. Open a new Chrome window
2. Click the Claude extension icon
3. Give Claude a task that involves the browser
4. Watch Claude navigate, click, scroll, and take screenshots

---

## Step 3: Run Multiple Instances

1. Open additional Chrome windows (not tabs — separate windows)
2. Start Claude in each window with a different task
3. Each instance runs independently

**Example multi-instance setup:**
| Window | Role | Task |
|--------|------|------|
| Window 1 | Researcher | Navigate Polymarket, find best "no" side bets |
| Window 2 | Coder | Build a backtest using backtesting.py |
| Window 3 | Analyst | Review algo trading roadmap, identify best strategies |

---

## Step 4: Design Your Prompt

Be specific about what you want Claude to do in the browser:

**Example prompt:**
> "Navigate to Polymarket. Go through the markets and find the top 10 where taking the 'no' side has the best chance of profiting. Send me the names, links, and why you think each is a good no market."

Claude will:
- Navigate to the website
- Scroll through content
- Take screenshots for analysis
- Follow multi-step instructions
- Log into accounts if needed

---

## When to Use Claude Chrome vs. OpenClaw

| Use Claude Chrome When... | Use OpenClaw When... |
|--------------------------|---------------------|
| Most work is in the browser | Need full computer control |
| Want the best model (Opus 4.6) | Want to use different/local models |
| Need smooth, integrated experience | Need deep customization |
| Working with web apps | Working with desktop apps |
| Running quick multi-instance tasks | Running persistent 24/7 agents |
