# Step-by-Step Guide: Building AI Teams with Claude Co-work Plugins

**Source:** Nicky Saunders (YouTube)
**Video ID:** ABh3KqQt8Tk
**Upload Date:** 2026-03-02

---

## What This Guide Covers

How to set up four AI "teams" (marketing, content repurposing, research, brand management) inside Claude's Co-work feature using plugins, skills, and scheduling — no employees needed.

---

## Prerequisites

- Claude Desktop App (download from claude.com)
- Paid Claude plan ($20+ /month)
- Claude Chrome Extension (for analytics automation)
- Brand documents folder (voice guides, company docs, etc.)

---

## Step 1: Install a Plugin

1. Open Claude Desktop App
2. Go to **Co-work** tab
3. Click **Customize**
4. Click **Browse Plugins**
5. Browse available departments (marketing, sales, legal, etc.)
6. Select a plugin to install (e.g., Marketing)

---

## Step 2: Customize Plugin to Your Brand

1. In the installed plugin, click **Customize**
2. Switch to the folder containing your brand documents
3. Include: brand voice guides, company docs, style guides, any reference materials
4. Hit **Let's Go**
5. Claude reads everything and auto-generates:
   - Commands (triggers for the plugin)
   - Skills (teams/workflows)
   - Connectors (app integrations)
6. Answer any follow-up questions Claude asks

---

## Step 3: Create a Content Repurposing Team

1. Go back to **Customize** on the marketing plugin
2. Enter a prompt like: "Customize the marketing plugin to repurpose my content into carousels, tweets, Substack, LinkedIn"
3. Claude creates a new **Content Creation Skill**
4. Test it: upload an old transcript or content piece
5. Claude outputs formatted content for all specified platforms in one document

---

## Step 4: Create a Research Agent

1. Customize the plugin again with a research-focused prompt
2. Specify: websites to monitor, influencers to track, topics of interest
3. Claude creates a **Research Skill** with all your sources
4. To automate: use `/schedule` command in Co-work
5. Configure:
   - Model to use
   - Frequency (daily, weekly, etc.)
   - Time to run
   - Reference folder
6. Research reports appear automatically on schedule

---

## Step 5: Create a Content Strategist (Auto-Analytics)

1. Customize the marketing plugin with a prompt like: "When I use the performance report, open the browser and find analytics directly — don't ask for screenshots"
2. This grants permission to use the **Claude Chrome Extension**
3. Requires: Claude Chrome Extension installed in your browser
4. Test: ask for an Instagram (or any platform) performance report
5. Claude opens Chrome, navigates to the platform, captures analytics
6. Produces a full report: reach, views, peak times, recommendations

## Tips

- Start with one department, get it working, then expand
- The more brand documents you provide, the better the customization
- You can have multiple plugins (departments) running simultaneously
- Use content prompts (linked in video description) to train Claude on your writing voice
- All of this works on the $20/month plan
