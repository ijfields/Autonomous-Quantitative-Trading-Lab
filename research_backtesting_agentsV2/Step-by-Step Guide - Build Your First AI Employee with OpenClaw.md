# Step-by-Step Guide: Building Specialist AI Employees with OpenClaw

---

## What This Guide Covers

How to create a team of specialized AI "employees" using a 5-question framework, with an orchestrator agent coordinating between them.

---

## Step 1: List Your Core Business Functions
1. Write down 3-4 areas where you spend the most time
2. For each area, identify recurring tasks that could be delegated
3. Start with the area that has the highest volume of repetitive work

## Step 2: Create Your First Specialist Agent
Use the 5-question framework:
1. **What is the specialty area?** (e.g., "X/Twitter content strategy")
2. **What is the specific objective?** (e.g., "grow engagement, increase followers, drive leads")
3. **What KPIs will this agent be rated on?** (e.g., "impressions, bookmarks, follower growth")
4. **What personality?** (e.g., "analytical but creative, studies competitor patterns")
5. **What tools/data does it need?** (e.g., "access to X API, competitor analysis, my posting history")

Then prompt: "Based on my answers, create a detailed agent persona with a name, emoji, role definition, daily responsibilities, success metrics, and communication style."

## Step 3: Set Up the Coordinator Agent
1. Create a "Chief of Staff" agent (e.g., "Max") as the single point of communication
2. All human communication goes through the coordinator
3. Specialists communicate with each other through the coordinator, not directly
4. The coordinator handles cross-domain decisions and information relay

## Step 4: Define Clear Boundaries
1. Each agent owns its own domain — no overlap
2. Define handoff processes between agents (when does work pass from one to another?)
3. Don't make specialists too narrow — "X content strategy with engagement focus" > "only write tweets about crypto"

## Step 5: Train and Iterate
1. Run the specialist for one full week before adding another
2. Provide feedback — agents improve through feedback loops
3. Adjust KPIs and responsibilities based on results
4. Only scale to the next specialist after the current one is reliable

## Step 6: Scale Your Team
1. Add specialists one at a time (recommended: no more than one new agent per week)
2. All agents can run on a single Claude Max subscription ($200/month)
3. Set up daily cron jobs for recurring tasks (e.g., 9:00 AM daily)
4. The human always approves big decisions — agents suggest, human decides, agents execute

---

## Key Takeaway

> "Specialists get better. Generalists stay where they are." Start with one specialist, master it, then scale. Use the 5-question framework for consistent agent creation. All agents run on a single $200/month subscription.
