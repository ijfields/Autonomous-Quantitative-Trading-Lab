## What This Guide Covers

How a non-developer uses Claude Code to build web apps, self-host services, and manage servers.

---

## Step 1: Set Up Your Development Environment

1. Install Claude Code (terminal-based) or use **Conductor** (GUI wrapper)
2. Conductor provides: repo view, chat interface, file diff view, terminal pane
3. No coding experience required — describe what you want in natural language

---

## Step 2: Build Your Application Locally

1. Describe your project to Claude Code
2. Iterate on the design and functionality through conversation
3. Test locally before deploying

---

## Step 3: Set Up Version Control + Deployment

1. Push code to **GitHub** repository
2. Connect GitHub to **Hostinger VPS** (KVM2 plan, ~$150/2 years)
3. Automatic deployment on every push

---

## Step 4: Self-Host Services via Claude Code

1. SSH into your VPS
2. Let Claude Code install and configure services:
   - **n8n** — automation platform (replaces Zapier/Make)
   - **Plausible Analytics** — privacy-friendly Google Analytics alternative
   - **OpenClaw** — AI agent platform via Docker
3. Claude Code handles Docker setup, configuration, and debugging

---

## Key Takeaway

> Claude Code enables non-developers to build complete web applications, self-host services, and manage servers through natural language conversation. Oliur built a membership site that earned $20,000 in its first week with zero prior coding experience.
