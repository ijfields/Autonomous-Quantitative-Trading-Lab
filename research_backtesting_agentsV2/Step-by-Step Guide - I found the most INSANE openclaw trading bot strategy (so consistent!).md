Recognize that the OpenClaw agent needs a specific strategy document (a "skill file" or "cartridge") to know how to trade. Without one, the agent has no trading rules.
- The skill file is a simple text/markdown file (skills.md) containing specific trading rules
- The agent uses these rules as its set of instructions for when and how to trade
- Think of it like a game cartridge that plugs into the back of the agent
- Expected result: You understand that the skill file is the key differentiator, not just the LLM

> Speaker's tip: "In the back of this bad boy the open claw agent is a slot and inside this slot you put a cartridge with a specific skill or specific strategy in it that they will use as a set of instructions and a set of rules."

## Common Pitfalls
- **Assuming any LLM will perform equally:** The race results show significant performance variance across LLMs. MiniMax excelled with the Surgeon, but OpenAI lost $3.20 in the first challenge with it. Always test.
- **Skipping the strategy/skill file:** You cannot just tell an AI agent "go trade." It needs specific, well-defined rules in a skill file to follow.
- **Over-relying on a single race result:** One profitable run could be a fluke. The speaker emphasizes running multiple challenges across multiple LLMs to validate consistency.
- **Using too much capital initially:** The races used $500 per agent. Start small and scale only after validating performance across multiple deployments.
- **Ignoring the entertainment disclaimer:** The speaker explicitly states this is for entertainment purposes. Crypto is highly volatile and the majority of retail clients will lose money.
