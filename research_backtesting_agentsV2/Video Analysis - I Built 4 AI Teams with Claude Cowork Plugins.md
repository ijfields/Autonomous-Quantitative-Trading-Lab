## EXECUTIVE SUMMARY

Nicky Saunders demonstrates how to use Claude's Co-work feature with plugins to create four AI "teams" — marketing, content repurposing, research, and brand management — all without employees or freelancers. He walks through installing plugins, customizing them with brand documents, creating skills (sub-teams), automating tasks with scheduling, and using the Claude Chrome extension to pull analytics automatically. Requires a paid Claude plan ($20+).

---

## KEY TOPICS

### Plugin Setup in Co-work
- Open Claude desktop app → Co-work → Customize → Browse Plugins
- Plugins cover departments: customer service, data, legal, sales, project management, marketing, etc.
- Think of plugins as departments; skills are smaller teams within each department

### Customizing Plugins to Your Brand
- Hit "Customize" on any plugin
- Point it to a folder with brand documents (brand voice, company docs, etc.)
- Claude reads all documents and auto-generates commands, skills, and connectors
- Result: plugin is tuned to your specific voice, community, and tools

### Creating Sub-Teams (Skills)
- **Content Repurposing Team:** Customize marketing plugin with prompt specifying platforms (carousels, tweets, Substack, LinkedIn)
- Claude creates a "content creation skill" that handles all platform formats
- Test by uploading an old transcript → outputs all content variants in one document

### Research Agent
- Create a research skill specifying websites, influencers, research sources
- Automate with `/schedule` command or the schedule menu
- Pick model, frequency, time, and reference folder
- Runs automatically and produces research reports

### Content Strategist (Analytics)
- Give Claude permission to use Chrome extension
- Instead of manually uploading screenshots, Claude opens browser and captures analytics directly
- Produces performance reports with reach, views, peak times, strategic recommendations

### Claude Chrome Extension
- Required for auto-analytics capture
- Claude takes over browser to navigate and find data
- Extension must be installed separately
