## What This Guide Covers
How to build a complete full-stack application using Orchids AI, from initial setup through iterative feature development, Claude Code integration, and project syncing between desktop and web.

---

## Step 1: Install Orchids Desktop Client
1. Go to orchids.app in your browser.
2. Log in to your account (or create one).
3. Click the download button in the upper right corner.
4. Choose your operating system (Windows, Mac, or other supported platform).
5. Download and install the application.
6. Open the desktop app and log in.

## Step 2: Generate the Front-End Structure
1. Start a new project in Orchids.
2. Prompt with a high-level description that generates structure only -- no real functionality yet.
3. Example prompt: "Create a simple mobile-optimized web app for [your idea]. Do not build the actual functionality yet. Just create placeholder data and sections so we have a better idea of how the app will look."
4. Wait for Orchids to generate the layout.
5. Review the visual structure -- you should see all major sections with placeholder data.
6. This step gives you direction before building real logic.

## Step 3: Build Core Features Iteratively
1. Focus on one feature at a time.
2. Example prompt: "Now please build the [feature name]. Allow users to [specific actions] and make the user experience as great as possible."
3. Wait for Orchids to process and generate the feature.
4. Test the feature by interacting with it (logging data, checking responses).
5. Verify the data is properly stored and displayed.
6. Repeat for each additional feature.

## Step 4: Add Predictive and Insight Features
1. Once basic tracking features work, build features that analyze existing data.
2. Example prompt: "Build the [insights feature]. Create automated insights based on the data users have already entered."
3. Verify that insights are dynamic and reflect real user data (not static placeholders).

## Step 5: Integrate Claude Code via Terminal (Desktop Only)
1. Inside the Orchids desktop app, add a terminal panel.
2. Type `claude` in the terminal (Claude Code must be installed on your machine).
3. Once Claude initializes, you can send it instructions within your project environment.
4. Example: "Assess the application, then create a dark theme and toggle button."
5. This lets you use your existing AI subscriptions (Claude, ChatGPT, etc.) at actual API cost -- no markup.

## Step 6: Sync Project from Desktop to Web
1. In the Orchids desktop app, click the three dots in the upper right corner.
2. Select "Sync."
3. Wait for the sync to complete.
4. Open your browser and access the project through the web platform.
5. Everything built locally is now available online.
6. Continue development from either environment.

## Step 7: Build Dashboard Widgets and Personalized Insights
1. Prompt Orchids to create dashboard widgets summarizing user activity across all features.
2. Example: "Finalize the app by creating widgets in the dashboard showing personalized insights for each functionality we have made."
3. Verify that widgets reflect data from each section and respond to newly logged data.

## Step 8: Final Review and Web Responsiveness
1. Ask Orchids to review the entire app for issues.
2. Prompt: "Check the app for any potential issues and create the proper web version that defaults to our current app's design when in mobile."
3. Verify the layout scales properly for web view.
4. Test the full user flow end-to-end: logging data, viewing dashboards, checking insights, toggling features.
