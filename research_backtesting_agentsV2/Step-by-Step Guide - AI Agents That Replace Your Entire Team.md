## What This Guide Covers

How to use the Agency Agents GitHub repository to deploy specialized AI agent personas for different business functions — from marketing to DevOps to legal.

---

## Prerequisites

- Claude, ChatGPT, or Cursor subscription
- GitHub account (to access the repository)
- Business context you want to apply agents to

---

## Step 1: Access the Repository

1. Go to the Agency Agents repository on GitHub (13K+ stars)
2. Browse the 51+ agent personas organized by business function
3. Identify which roles are most relevant to your needs

---

## Step 2: Choose Your Agent Personas

| If You Need... | Use This Agent |
|----------------|---------------|
| Go-to-market strategy | Marketing Director |
| Sales process improvement | Sales Manager |
| Code quality auditing | Technical Debt Specialist |
| Customer retention strategy | Customer Success Manager |
| Product roadmap planning | Product Manager |
| Infrastructure decisions | DevOps Engineer |
| PR crisis handling | Crisis Communication Specialist |
| AI implementation ethics | AI Ethics Adviser |
| Financial modeling | Financial Analyst |
| Legal review | Legal Affairs Specialist |

---

## Step 3: Load the Persona

1. Copy the agent persona prompt from the repository
2. Paste it as a system prompt in Claude, ChatGPT, or Cursor
3. Interact with the agent as if they were a real team member in that role
4. Ask role-specific questions, not generic ones

**Example:** Instead of "Help me with marketing," say: "I'm launching a new SaaS product targeting mid-market companies. What should my go-to-market strategy be?"

---

## Step 4: Customize for Your Context

1. Add your industry-specific context to the persona prompt
2. Include company details, target market, and constraints
3. Modify the agent's priorities to match your business stage
4. Save customized versions for reuse

---

## Step 5: Build a Multi-Agent Team

1. Load different personas for different business functions
2. Run each in its own conversation/window
3. Cross-reference advice between agents (e.g., marketing strategy → check with financial analyst)
4. The more specific each agent's scope, the better the output

---

## Key Takeaway

> Specialized AI agents with clear role definitions dramatically outperform generic AI assistants. A $20/month AI subscription gives a solo founder access to the equivalent of a full executive team — if you use the right prompts.
