## What This Guide Covers
Five GitHub repositories that enhance Claude Code workflows, covering design systems, project scaffolding, n8n automation, Obsidian knowledge management, and curated tooling collections. (Note: despite the video title, the transcript covers Claude Code repos, not Blender or OpenClaw.)

## Step 1: Install UIUX Promax for Intelligent Design Systems
1. Find the UIUX Promax GitHub repository
2. Clone or download it into your Claude Code project environment
3. Use it when building front-end interfaces to generate context-aware design systems
4. Benefit: produces design output tailored to what you are building, avoiding generic AI-generated UI patterns

## Step 2: Set Up GSD (Get Stuff Done) Framework
1. Locate the GSD GitHub repository
2. Integrate it on top of Claude Code as a spec-driven development framework
3. Use it when starting a new project from scratch
4. It automatically handles sub-agent coordination and context window management ("context window rot")
5. Write specifications first, then let GSD drive the development process

## Step 3: Add the n8n MCP Server (When Needed)
1. Find the n8n MCP Server GitHub repository
2. Install and configure it as an MCP server for Claude Code
3. This gives Claude Code access to 1,000+ n8n nodes with their properties, operations, and documentation
4. It also includes nearly 3,000 real-world workflow examples
5. Note: this MCP server is resource-heavy -- only use it when your project specifically targets n8n workflow automation
6. Best for projects where the end goal involves a complex n8n solution

## Step 4: Integrate Obsidian Skills for a Second Brain
1. Find the Obsidian Skills GitHub repository
2. Set it up to connect Claude Code with an Obsidian vault
3. This creates a persistent second brain that remembers everything across sessions
4. Use it for knowledge management, note-taking, and maintaining context across long-running projects

## Step 5: Bookmark Awesome Claude Code as a Reference
1. Find the Awesome Claude Code GitHub repository
2. Star/bookmark it for ongoing reference
3. Browse the collection of skills, workflows, toolings, and commands
4. Use it as a one-stop directory whenever you need to find new Claude Code extensions or capabilities

> Five community-built GitHub repos can dramatically extend Claude Code's capabilities: UIUX Promax for design, GSD for project scaffolding, n8n MCP Server for automation workflows, Obsidian Skills for persistent memory, and Awesome Claude Code for a curated tooling directory.
